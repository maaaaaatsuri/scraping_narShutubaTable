1. 予想する

[ HOME ] -> [ 最近のレース ] から対象レースを選択します。馬名、騎手、厩舎のいずれかの条件を指定することで、並び替えができます。
※ 条件を指定せずに検索することで、並び替えをリセットします。

また、[ HOME ] -> [ その他のレース ] からは、過去のレース結果などを検索することができます。


2. 編集する

[ HOME ] -> [ レースを編集する ] から編集項目を選択します。主に過去のレースを削除したり、新規レースを取得するのに使用します。
※ アプリ起動中は、当日と翌日のレースを一定時間おきに自動取得しています。
