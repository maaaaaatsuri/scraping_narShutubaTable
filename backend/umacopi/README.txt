アプリ名：
  『UmaCopi(うまコピ)』（地方競馬出馬表スクレイピングアプリ）

概要：
  WEBサイト "https://nar.netkeiba.com/top/" から、情報を自動取得する【出馬表スクレイピングアプリ】です。

説明：
　・1回/30分、当日の出馬表及びレース結果を自動取得します。
　・1回/3時間、翌日の情報を自動取得します。
  ・出走馬ごとの情報がDBに格納されているので、任意の条件で並び替えが可能です。

機能：
  ・WEBサイト "https://nar.netkeiba.com/top/" で公開されているレース情報を自動取得する。
  ・取得した情報を「馬名」「騎手」「厩舎」で並べ替える。


使用例：
```````````````````````````````````````````````````````````
  $ git clone https://github.com/maaaaaatsuri/umacopi.git
  $ cd /root/umacopi/backend/umacopi/umacopi_pack
  $ pip3 install -r requirements.txt
```````````````````````````````````````````````````````````

その他：
  出馬表を並び替えて、予想をもっと楽にしよう。
